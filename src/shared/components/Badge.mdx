# Badge

상태, 중요도, 라벨 등을 표시하는 인라인 배지 컴포넌트입니다.

## 변형 (Variants)

### 상태 배지 (Status)

이슈의 현재 상태를 나타냅니다. `variant="status"`와 `status` prop을 함께 사용합니다.

| 상태         | 라벨     | 색상    |
| ------------ | -------- | ------- |
| `ongoing`    | 진행중   | danger  |
| `closed`     | 종결     | success |
| `reignited`  | 재점화   | warning |
| `unverified` | 확인필요 | primary |

<Canvas of={BadgeStories.AllStatuses} />

### 중요도 배지 (Importance)

`variant="importance"`와 `importance` prop을 함께 사용합니다.

<Canvas of={BadgeStories.AllImportances} />

### 라벨 배지 (Breaking / Urgent / New)

뉴스성 라벨을 표시합니다. children으로 라벨을 커스텀할 수 있으며, 미지정 시 기본 라벨이 표시됩니다.

<Canvas of={BadgeStories.LabelBadges} />

### 태그 배지 (Tag)

분류 태그를 표시합니다. children으로 태그명을 전달합니다.

<Canvas of={BadgeStories.Tag} />

### 기본 배지 (Default)

범용 배지입니다. children으로 텍스트를 전달합니다.

<Canvas of={BadgeStories.Default} />

## 접근성

- `variant="status"` 사용 시 `role="status"`와 `aria-label`이 자동 설정됩니다.
- `variant="importance"` 사용 시 `aria-label`이 자동 설정됩니다.
