# Input

기본 텍스트 입력 컴포넌트입니다. 2가지 변형(variant)과 3가지 크기(size)를 지원합니다.

## 변형 (Variants)

| 변형      | 설명                               |
| --------- | ---------------------------------- |
| `default` | 테두리 스타일 (기본값)             |
| `filled`  | 배경 채움 스타일, 포커스 시 테두리 |

<Canvas of={InputStories.AllVariants} />

## 크기 (Sizes)

| 크기 | 높이 | 용도              |
| ---- | ---- | ----------------- |
| `sm` | 32px | 좁은 공간         |
| `md` | 40px | 기본 (기본값)     |
| `lg` | 48px | 강조, 모바일 터치 |

import { Input } from "../shared/components";
