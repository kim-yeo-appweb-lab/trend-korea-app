# Card

Compound 패턴을 사용하는 카드 컴포넌트입니다. 하위 컴포넌트를 자유롭게 조합하여 다양한 레이아웃을 구성할 수 있습니다.

## 하위 컴포넌트

| 컴포넌트      | 설명                              |
| ------------- | --------------------------------- |
| `Card`        | 카드 컨테이너 (border + surface)  |
| `CardHeader`  | 상단 영역 (제목 + 액션)           |
| `CardTitle`   | 제목 텍스트                       |
| `CardContent` | 본문 영역                         |
| `CardFooter`  | 하단 영역 (메타 정보, border-top) |

## 전체 조합

```tsx
<Card>
	<CardHeader>
		<CardTitle>제목</CardTitle>
		<span>액션</span>
	</CardHeader>
	<CardContent>본문</CardContent>
	<CardFooter>
		<span>날짜</span>
		<span>댓글 수</span>
	</CardFooter>
</Card>
```
