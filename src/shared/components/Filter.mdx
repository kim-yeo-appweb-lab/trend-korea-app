## 사용법

```tsx
import { Filter, FilterGroup } from "../shared/components";

// 타입 안전한 필터 (제네릭)
type Period = "all" | "today" | "week" | "month";

const options: { value: Period; label: string }[] = [
	{ value: "all", label: "전체" },
	{ value: "today", label: "오늘" },
	{ value: "week", label: "이번 주" },
	{ value: "month", label: "이번 달" }
];

<Filter<Period> label="기간" options={options} value={selected} onChange={setSelected} />;
```

## 접근성

- `role="radiogroup"` / `role="radio"` 자동 적용
- `aria-checked`로 선택 상태 표시
- `aria-label`로 필터 그룹 설명 제공
