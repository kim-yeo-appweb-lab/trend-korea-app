# SectionHeader

페이지 내 섹션의 제목과 "더보기" 링크를 표시하는 헤더 컴포넌트입니다.

## 기본 사용법

<Canvas of={SectionHeaderStories.Default} />

## 더보기 링크 포함

`href`를 전달하면 우측에 "더보기" 링크가 표시됩니다. `linkLabel`로 텍스트를 커스텀할 수 있습니다.

<Canvas of={SectionHeaderStories.WithLink} />

## children으로 추가 요소 배치

제목과 링크 사이에 배지 등 추가 요소를 배치할 수 있습니다.

<Canvas of={SectionHeaderStories.WithChildren} />

## 사용법

```tsx
import { SectionHeader } from "../shared/components";

// 기본
<SectionHeader title="실시간 이슈" />

// 더보기 링크 포함
<SectionHeader title="실시간 이슈" href="/issues" />

// children으로 배지 추가
<SectionHeader title="인기 게시글" href="/community">
  <Badge variant="new" />
</SectionHeader>
```
