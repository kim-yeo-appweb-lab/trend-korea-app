# Typography

Geist Sans 기반 타이포그래피 스케일입니다. 15단계로 구성되며 각 단계는 고유한 font-size, line-height, font-weight 조합을 가집니다.

---

## Display

페이지 상단, 히어로 섹션 등 대형 제목에 사용합니다.

<Typeset
	fontSizes={["36px", "30px"]}
	fontWeight={700}
	sampleText="트렌드 코리아"
	fontFamily="var(--font-geist-sans), system-ui, sans-serif"
/>

| 토큰         | font-size | line-height | font-weight |
| ------------ | --------- | ----------- | ----------- |
| `display-lg` | 36px      | 44px        | 700 (Bold)  |
| `display-md` | 30px      | 38px        | 700 (Bold)  |

---

## Heading

섹션 제목, 카드 헤더 등에 사용합니다.

<Typeset
	fontSizes={["24px", "20px", "18px", "16px"]}
	fontWeight={600}
	sampleText="실시간 이슈 타임라인"
	fontFamily="var(--font-geist-sans), system-ui, sans-serif"
/>

| 토큰         | font-size | line-height | font-weight    |
| ------------ | --------- | ----------- | -------------- |
| `heading-xl` | 24px      | 32px        | 700 (Bold)     |
| `heading-lg` | 20px      | 28px        | 600 (SemiBold) |
| `heading-md` | 18px      | 26px        | 600 (SemiBold) |
| `heading-sm` | 16px      | 24px        | 600 (SemiBold) |

---

## Title

리스트 항목 제목, 네비게이션 레이블 등에 사용합니다.

<Typeset
	fontSizes={["16px", "14px", "12px"]}
	fontWeight={500}
	sampleText="대한민국 사건 아카이브"
	fontFamily="var(--font-geist-sans), system-ui, sans-serif"
/>

| 토큰       | font-size | line-height | font-weight  |
| ---------- | --------- | ----------- | ------------ |
| `title-lg` | 16px      | 24px        | 500 (Medium) |
| `title-md` | 14px      | 20px        | 500 (Medium) |
| `title-sm` | 12px      | 16px        | 500 (Medium) |

---

## Body

본문 텍스트, 설명, 기사 내용 등에 사용합니다.

<Typeset
	fontSizes={["16px", "14px", "12px"]}
	fontWeight={400}
	sampleText="2024년 주요 사건과 이슈를 시간순으로 추적하고 아카이빙합니다. 실시간 업데이트와 커뮤니티 토론을 통해 사회적 흐름을 파악할 수 있습니다."
	fontFamily="var(--font-geist-sans), system-ui, sans-serif"
/>

| 토큰      | font-size | line-height | font-weight   |
| --------- | --------- | ----------- | ------------- |
| `body-lg` | 16px      | 28px        | 400 (Regular) |
| `body-md` | 14px      | 24px        | 400 (Regular) |
| `body-sm` | 12px      | 20px        | 400 (Regular) |

---

## Label

버튼, 배지, 폼 레이블, 캡션 등에 사용합니다.

<Typeset
	fontSizes={["14px", "12px", "11px"]}
	fontWeight={500}

	fontFamily="var(--font-geist-sans), system-ui, sans-serif"

| 토큰       | font-size | line-height | font-weight  |
| ---------- | --------- | ----------- | ------------ |
| `label-lg` | 14px      | 20px        | 500 (Medium) |
| `label-md` | 12px      | 16px        | 500 (Medium) |
| `label-sm` | 11px      | 16px        | 500 (Medium) |

## 사용 가이드

### 조합 예시

```
Display-lg (36/44/700)  → 히어로 제목
Heading-lg (20/28/600)  → 카드 제목
Body-md (14/24/400)     → 카드 본문
Label-md (12/16/500)    → 배지, 태그
```

### 접근성

- 본문 텍스트 최소 크기: `body-sm` (12px)
- 행간(line-height)은 WCAG 2.1 기준 최소 1.5배 이상 유지
- `body-lg`(16px/28px = 1.75)와 `body-md`(14px/24px = 1.71)는 가독성 최적화
