# Spacing

4px 기반 스페이싱 스케일입니다. 일관된 간격으로 시각적 리듬을 만들어 냅니다.

---

## 스케일

<div style={{ display: "flex", flexDirection: "column", gap: "12px", marginTop: "16px" }}>
	{[
		{ token: "space-0", value: "0px", px: 0 },
		{ token: "space-1", value: "4px", px: 4 },
		{ token: "space-2", value: "8px", px: 8 },
		{ token: "space-3", value: "12px", px: 12 },
		{ token: "space-4", value: "16px", px: 16 },
		{ token: "space-5", value: "20px", px: 20 },
		{ token: "space-6", value: "24px", px: 24 },
		{ token: "space-7", value: "28px", px: 28 },
		{ token: "space-8", value: "32px", px: 32 },
		{ token: "space-9", value: "36px", px: 36 },
		{ token: "space-10", value: "40px", px: 40 },
		{ token: "space-12", value: "48px", px: 48 },
		{ token: "space-14", value: "56px", px: 56 },
		{ token: "space-16", value: "64px", px: 64 },
		{ token: "space-20", value: "80px", px: 80 },
		{ token: "space-24", value: "96px", px: 96 }
	].map((item) => (
		<div key={item.token} style={{ display: "flex", alignItems: "center", gap: "16px" }}>
			<code style={{ width: "100px", fontSize: "13px", fontFamily: "monospace", flexShrink: 0 }}>{item.token}</code>
			<span style={{ width: "50px", fontSize: "13px", color: "#71717a", flexShrink: 0 }}>{item.value}</span>
			<div
				style={{
					height: "24px",
					width: `${Math.max(item.px, 1)}px`,
					background: "#ff6b47",
					borderRadius: "4px",
					minWidth: item.px === 0 ? "1px" : undefined,
					opacity: item.px === 0 ? 0.3 : 1
				}}
			/>
		</div>
	))}
</div>

## Tailwind 사용법

Tailwind CSS 4에서 `--spacing-*` 토큰이 자동 등록됩니다.

```html
<!-- 패딩 -->
<div class="p-4">16px 패딩</div>
<div class="px-6 py-3">수평 24px, 수직 12px</div>

<!-- 마진 -->
<div class="mt-8">상단 32px 마진</div>

<!-- 갭 -->
<div class="flex gap-2">아이템 간 8px 간격</div>
<div class="grid gap-6">그리드 24px 간격</div>
```

## 사용 가이드

### 권장 매핑

| 용도               | 토큰                    | 값      |
| ------------------ | ----------------------- | ------- |
| 아이콘-텍스트 간격 | `space-1` ~ `space-2`   | 4~8px   |
| 요소 내부 패딩     | `space-2` ~ `space-4`   | 8~16px  |
| 카드 패딩          | `space-4` ~ `space-6`   | 16~24px |
| 섹션 내 간격       | `space-6` ~ `space-8`   | 24~32px |
| 섹션 간 간격       | `space-10` ~ `space-16` | 40~64px |
| 페이지 상하 여백   | `space-16` ~ `space-24` | 64~96px |

### 핵심 원칙

1. **4의 배수**: 모든 간격은 4px의 배수를 사용합니다
2. **시각적 계층**: 관련 요소는 가까이, 독립 요소는 멀리 배치합니다
3. **일관성**: 동일한 맥락에서는 동일한 토큰을 사용합니다
