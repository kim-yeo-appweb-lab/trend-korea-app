# Shadows

5단계(level 0~4) 그림자 시스템입니다. 요소의 수직적 계층(elevation)을 시각적으로 표현합니다.
라이트/다크 모드에서 불투명도를 달리하여 적절한 깊이감을 유지합니다.

---

## 라이트 모드

<div
	style={{
		display: "flex",
		flexDirection: "column",
		gap: "24px",
		marginTop: "16px",
		padding: "32px",
		background: "#fafafa",
		borderRadius: "12px"
	}}
>
	{[
		{ level: 0, shadow: "none", desc: "그림자 없음" },
		{ level: 1, shadow: "0 1px 2px 0 rgb(0 0 0 / 0.05)", desc: "미세한 깊이" },
		{ level: 2, shadow: "0 1px 3px 0 rgb(0 0 0 / 0.1), 0 1px 2px -1px rgb(0 0 0 / 0.1)", desc: "카드 기본" },
		{ level: 3, shadow: "0 4px 6px -1px rgb(0 0 0 / 0.1), 0 2px 4px -2px rgb(0 0 0 / 0.1)", desc: "드롭다운, 팝오버" },
		{ level: 4, shadow: "0 10px 15px -3px rgb(0 0 0 / 0.1), 0 4px 6px -4px rgb(0 0 0 / 0.1)", desc: "모달, 다이얼로그" }

		<div key={item.level} style={{ display: "flex", alignItems: "center", gap: "20px" }}>

					boxShadow: item.shadow,

					alignItems: "center",

				<span style={{ fontSize: "14px", fontWeight: 500, color: "#18181b" }}>Level {item.level}</span>
			</div>

				<p style={{ fontSize: "12px", color: "#71717a", margin: "4px 0 0" }}>{item.desc}</p>
			</div>
		</div>
	))}
</div>

## 레퍼런스 테이블

| 토큰      | 라이트 모드                         | 다크 모드                           | 용도        |
| --------- | ----------------------------------- | ----------------------------------- | ----------- |
| `level-0` | `none`                              | `none`                              | 플랫 요소   |
| `level-1` | `0 1px 2px rgb(0 0 0 / 0.05)`       | `0 1px 2px rgb(0 0 0 / 0.3)`        | 미세한 깊이 |
| `level-2` | `0 1px 3px rgb(0 0 0 / 0.1), ...`   | `0 1px 3px rgb(0 0 0 / 0.4), ...`   | 카드 기본   |
| `level-3` | `0 4px 6px rgb(0 0 0 / 0.1), ...`   | `0 4px 6px rgb(0 0 0 / 0.4), ...`   | 드롭다운    |
| `level-4` | `0 10px 15px rgb(0 0 0 / 0.1), ...` | `0 10px 15px rgb(0 0 0 / 0.5), ...` | 모달        |

## 사용 가이드

### Tailwind 클래스

```html
<!-- 카드 -->
<div class="shadow-level-1 bg-surface rounded-lg p-4">기본 카드</div>

<!-- 호버 시 깊이 추가 -->
<div class="shadow-level-1 hover:shadow-level-2 duration-fast transition-shadow">호버 효과</div>

<!-- 드롭다운 -->
<div class="shadow-level-3 bg-elevated rounded-md">드롭다운 메뉴</div>

<!-- 모달 -->
<div class="shadow-level-4 bg-surface rounded-lg">모달 컨텐츠</div>
```

### 권장 매핑

| 컴포넌트                 | 그림자 레벨 |
| ------------------------ | ----------- |
| 인라인 요소, 플랫 UI     | `level-0`   |
| 카드, 리스트 아이템      | `level-1`   |
| 호버 상태, 선택된 카드   | `level-2`   |
| 드롭다운, 팝오버, 툴팁   | `level-3`   |
| 모달, 대화상자, 바텀시트 | `level-4`   |
