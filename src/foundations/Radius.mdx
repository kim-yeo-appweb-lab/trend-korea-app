# Border Radius

7단계 라운딩 스케일로 요소의 형태감을 제어합니다.

---

## 스케일

<div style={{ display: "flex", flexWrap: "wrap", gap: "24px", marginTop: "16px" }}>
	{[
		{ token: "xs", value: "2px", radius: 2 },
		{ token: "sm", value: "4px", radius: 4 },
		{ token: "md", value: "8px", radius: 8 },
		{ token: "lg", value: "12px", radius: 12 },
		{ token: "xl", value: "16px", radius: 16 },
		{ token: "2xl", value: "24px", radius: 24 },
		{ token: "full", value: "9999px", radius: 9999 }
	].map((item) => (
		<div key={item.token} style={{ display: "flex", flexDirection: "column", alignItems: "center", gap: "8px" }}>
			<div
				style={{
					width: "80px",
					height: "80px",
					borderRadius: `${item.radius}px`,
					background: "#ff6b47",
					opacity: 0.85
				}}
			/>
			<code style={{ fontSize: "13px", fontFamily: "monospace" }}>radius-{item.token}</code>
			<span style={{ fontSize: "12px", color: "#71717a" }}>{item.value}</span>
		</div>
	))}
</div>

---

## 레퍼런스 테이블

| 토큰   | CSS 변수        | 값       | Tailwind 클래스 | 용도                     |
| ------ | --------------- | -------- | --------------- | ------------------------ |
| `xs`   | `--radius-xs`   | `2px`    | `rounded-xs`    | 미세한 라운딩, 코드 블록 |
| `sm`   | `--radius-sm`   | `4px`    | `rounded-sm`    | 입력 필드, 작은 버튼     |
| `md`   | `--radius-md`   | `8px`    | `rounded-md`    | 카드, 버튼, 드롭다운     |
| `lg`   | `--radius-lg`   | `12px`   | `rounded-lg`    | 모달, 대화상자           |
| `xl`   | `--radius-xl`   | `16px`   | `rounded-xl`    | 큰 카드, 패널            |
| `2xl`  | `--radius-2xl`  | `24px`   | `rounded-2xl`   | 특수 요소, 풀 카드       |
| `full` | `--radius-full` | `9999px` | `rounded-full`  | 원형 (아바타, 배지, 필)  |

| 큰 카드, 패널          | `xl` (16px)     |
| 아바타, 상태 배지, 필  | `full` (9999px) |
