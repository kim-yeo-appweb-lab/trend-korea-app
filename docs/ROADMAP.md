# ROADMAP (프론트엔드)

> PRD 기반 프론트엔드 개발 로드맵
> 생성일: 2026-03-09
> 기반 PRD: `trend-korea-app/docs/PRD.md`, `docs/PRD.md`

## 개요

대한민국 사건/이슈를 시간순으로 기록하고 지속 추적하는 정보 아카이브 서비스의 **Next.js 프론트엔드** 로드맵.

### 기술 스택

| 레이어        | 기술                                                 |
| ------------- | ---------------------------------------------------- |
| 프레임워크    | Next.js 16 (App Router) + React 19                   |
| 언어          | TypeScript 5.9 (strict mode)                         |
| 스타일링      | Tailwind CSS 4                                       |
| UI 라이브러리 | @kim-yeo-appweb-lab/ui (shadcn/ui 마이그레이션 예정) |
| 데이터 페칭   | @tanstack/react-query v5 + ky                        |
| 폼            | react-hook-form v7 + zod v4                          |
| 테스트        | vitest + @testing-library/react + msw                |
| 린팅/포맷팅   | ESLint 9 (flat config) + Prettier + lefthook         |
| 패키지 매니저 | pnpm                                                 |

### 테스트 전략

- **프레임워크**: vitest + @testing-library/react + msw (API 목킹)
- **테스트 트로피**: Static Analysis 20% / Unit 30% / Integration 40% / E2E 10%
- **Phase 3 이후**: TDD 필수 (Red-Green-Refactor)
- **Static Analysis**: TypeScript strict mode + ESLint 9 + Prettier

### 현재 상태

MVP 사용자 서비스 프론트엔드 완료 (8개 feature 모듈, 15개 페이지, 17개 React Query 훅, 6개 Zod 스키마). 어드민 UI 미구현.

> **참고**: 어드민 기능은 독립 프로젝트 `trend-korea-admin`으로 분리되었습니다.
> Phase 1~3 및 Phase 4-1~4-4의 어드민 관련 태스크는 해당 프로젝트로 이관되었습니다.
